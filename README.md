# PlayStop Frontend

| Rol         | Acceso                                      |
|-------------|---------------------------------------------|
| JUGADOR     | Reservar canchas, ver historial             |
| PROPIETARIO | Gestionar canchas propias, ver reservas     |
| ADMIN       | Administrar instalaciones asignadas         |
| SUPER ADMIN | Control total de la plataforma              |
